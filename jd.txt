Job Summary:
Senior backend engineer role driving scalable, high‑performance systems for web and game-related platforms using modern technologies.

Responsibilities:
Develop and maintain web-based and game-related applications using modern frameworks such as Ruby on Rails and C# .NET.
Write, test, and optimise code in JavaScript and Python to support backend services and game features.
Collaborate with cross-functional teams using Git-based version control to deliver high-quality solutions.
Manage application dependencies and libraries using NPM, PIP, and Ruby Gems.
Design, develop, and maintain front-end components using ReactJS for interactive user interfaces.
Work with SQL and NoSQL databases to store, retrieve, and manage application and game data.
Collaborate with designers, developers, and product stakeholders to deliver engaging and interactive solutions.

Requirements:
Minimum 3 years of relevant experience in Ruby on Rails development.
Experience with modern web development framework such as Ruby on Rails, C# .NET.
Proficiency in JavaScript and Python for web application and service development.
Strong understanding of Git and version control workflows.
Familiarity with package management tools including NPM, PIP, and Ruby Gems.
Knowledge of SQL and NoSQL databases such as MongoDB, MySQL, or MSSQL is an added advantage.
Experience with ReactJS or other JavaScript frameworks is an added advantage.
Background in game development or experience building gaming-related applications is an added advantage.
